Please enter the uncertainty adjustment factors, correct to 4 decimal places.
You can find examples of how to calculate these in Guidance Note 19.

The individual adjusment factor (FI) is given by:

For energy inputs:
So if the UX = 1.50%,

FI = 1 + (excess uncertainty (UX) / 100)
FI = 1 + (1.50 / 100) = 1.0150

For power or heat outputs:
So if the UX = 1.50%,

FI = 1 - (UX / 100)
FI = 1 - (1.50 / 100) = 0.9850
